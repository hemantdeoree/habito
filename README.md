# Habit and Task Tracker

A simple web-based habit and task tracker that helps you monitor your daily, monthly, and yearly progress.

## Features

- Add habits to track
- Mark habits as completed each day
- View progress:
  - Daily: Number of habits completed today
  - Monthly: Total completions this month
  - Yearly: Total completions this year

## How to Use

1. Open `index.html` in your web browser.
2. Add a new habit by entering its name and clicking "Add Habit".
3. Each day, check the boxes for habits you've completed.
4. View your progress in the Progress section.

## Data Storage

Your habits and completion data are stored locally in your browser's localStorage, so your data persists between sessions.
